## Анализ ARP-сообщений

**Очистка ARP-кэша на Kali**

ARP-кэш содержит соответствия между IP-адресами и MAC-адресами устройств в локальной сети. Для того чтобы гарантированно увидеть широковещательный ARP-запрос, кэш был очищен командой:

```bash
sudo ip neigh flush all
```

После выполнения команды таблица ARP стала пустой – при следующей отправке пакета на любой IP-адрес в подсети Kali будет вынуждена выполнить ARP-запрос.

**Отправка ICMP-запроса**

Для инициирования ARP-обмена и последующего ICMP-трафика был отправлен один ping на адрес шлюза (192.168.47.1):

```bash
ping -c 1 192.168.47.1
```

  - Сообщает: «IP 192.168.47.1 принадлежит MAC 00:0c:29:ca:61:95».

## Разбор TCP-сессии (netcat)

В связи с отсутствием утилиты `netcat` на Windows Desktop (команда `nc` не распознана), в качестве сервера используется Ubuntu Router (192.168.47.1). Трафик между Kali и роутером будет проходить через интерфейс `eth2` Kali, что позволит Wireshark перехватить все пакеты TCP-сессии.

**Запуск сервера на Ubuntu Router**

```bash
nc -l -p 12345
```

Роутер переходит в режим ожидания входящих TCP-соединений. Терминал блокируется до появления клиента.

**Включение фильтра Wireshark на Kali**

```
tcp.port == 12345
```

**Подключение клиента с Kali**

На Kali во втором терминале инициируется подключение к серверу на роутере:

```bash
nc 192.168.47.1 12345
```

## Перехват пароля через HTTP-аутентификацию


### Задача

На Kali запустить веб-сервер, принимающий POST-запросы с формой авторизации. С Windows Desktop открыть страницу, ввести логин и пароль. В Wireshark найти HTTP-пакет, содержащий введённые данные.

**Создание файла с веб-приложением**

Файл `app.py`:

```python
from flask import Flask, request

app = Flask(__name__)

@app.route('/')
def form():
    return '''
    <form method="POST" action="/login">
        <label>Login:</label>
        <input type="text" name="user"><br>
        <label>Password:</label>
        <input type="password" name="pass"><br>
        <input type="submit" value="Login">
    </form>
    '''

@app.route('/login', methods=['POST'])
def login():
    user = request.form.get('user')
    pwd = request.form.get('pass')
    print(f"Received: {user}:{pwd}")
    return f"Hello {user}, your password is {pwd} (captured by Wireshark)."

if __name__ == '__main__':
    app.run(host='0.0.0.0', port=8080)
```

**Запуск веб-сервера**

```bash
python3 app.py
```

Сервер начинает работу на порту 8080, прослушивая все сетевые интерфейсы.

**Настройка фильтра в Wireshark**

```
http
```

**Отправка POST-запроса с Windows Desktop**

На машине Windows Desktop открывается веб-браузер. В адресной строке вводится:

```
http://192.168.47.52:8080/
```

После загрузки страницы в поля формы вводятся учётные данные пользователя:

- Логин: `student`
- Пароль: `12345`
